# 🎰 EpiBet — Plateforme temps réel

## 🧠 Concept
Les jeux envoient leurs transactions à une API centrale.

## 🏗️ Architecture
[ Jeux ] ---> [ API ] ---> [ Dashboard ]

## ⚙️ Stack
Backend: Node.js, Express, SQLite  
Frontend: React, Vite

## 🚀 Lancement
docker compose up --build

## 🌐 Accès
Front: http://localhost:3000  (ou l'ip hebergeur)
API: http://localhost:8000 (ou l'ip hebergeur)

## 🎮 Fonctionnement
- register
- create game
- send transactions

## 🔐 Sécurité
Clé API, aucune donnée joueur

## 💬 Pitch
API temps réel pour jeux connectés avec dashboard live.
